# Gaming Tournament Platform Database

## Overview
This project was developed as part of a university database course.
The goal was to design and implement a relational database for a gaming tournament platform using SQL and Vertabelo.

## Features
- Relational database design
- Database schema created in Vertabelo
- 10+ normalized tables
- SQL scripts for database creation
- SQL scripts for data population
- Complex SQL queries using joins and subqueries
## Technologies
- SQL
- Vertabelo
## Repository Structure

- `Gaming Tournament_Platform create.sql` – database schema
- `Gaming Tournament DML.sql` – sample data
- `queries.sql` – SQL queries
- `Gaming_Tournament_Platform.png` – ER diagram
- `Gaming_Tournament_Platform.xml` – Vertabelo project
